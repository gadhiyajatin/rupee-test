# **App Name**: "":upee Book

## Core Features:

- Multiple Cashbooks: Users can create multiple cashbooks (e.g., "Personal Book", "Business Book")
- Cashbook System: Cash In & Cash Out entries with manual categories (user-defined only)
- Transaction Filters: Transaction filtering by date range, type (In/Out), party, category, and payment mode
- Transaction Search: Search transactions by amount or remark
- Field Customization: Rename, add, delete, reorder, or hide party, category, payment mode, and custom fields

## Style Guidelines:

- Light Blue (#ADD8E6) to evoke trust and reliability in handling finances.
- Very light Blue (#F0F8FF), creating a clean and calming backdrop.
- Pale Green (#98FB98), complementing the primary color and signaling income.
- Font: 'Mukta', a Devanagari and Latin typeface, ensuring readability in both English and Gujarati.
- Use a clear, legible font size optimized for both English and Gujarati scripts, ensuring accessibility for all users.
- Incorporate icons that resonate with Indian culture and financial practices, using a minimalist line-style for clarity.
- Design a clean, tabular layout for displaying transaction data, optimized for both mobile and web interfaces, with consideration for right-to-left text in Gujarati.
- Implement subtle animations on transaction entry and filtering to provide feedback and enhance user experience, with consideration for cultural preferences in visual feedback.
